3. RISK MANAGEMENT - Kelly Criterion
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   ✅ js/kelly_calculator.js                (420 lines)
      • Kelly Criterion calculator
      • Functions:
        - calculateKellyOptimal() → Full Kelly %
        - calculateKellyFractional() → Safe Kelly (1/2, 1/4, etc.)
        - calculateBetSize() → Currency amount
        - calculateExpectedValue() → EV calculation
        - calculateEdge() → Model vs bookmaker
        - calculateRiskOfRuin() → Bankruptcy probability
        - evaluateBet() → Full evaluation with warnings
        - calculateCombinedKelly() → Correlated bets
      • Risk warnings generated for:
        - No edge detected
        - Small edge (<1.5%)
        - Stake above Kelly
        - High risk of ruin (>5%)
        - Excessive stake (>10% bankroll)
      • Returns recommendations based on metrics

                         API ADDITIONS
================================================================================

New Endpoints:

1. POST /api/chat
   Request:  { message, session_id, language, bankroll }
   Response: { response, tool_calls[], recommendations[], confidence }
   Speed:    ~500ms (Groq Mixtral model)

2. GET /api/odds
   Request:  ?match_id=ARG-MAR&market=1x2&model_prob=0.45
   Response: { odds, edge_pct, recommendation, implied_probability }
   Cache:    5 minutes (in-memory)

Enhanced Endpoints:

3. POST /api/bets (can now store Kelly %)
   New fields: kelly_pct, edge_calc, expected_value

4. GET /api/learn (enhanced Tavily research)
   Now uses 8 searches/day instead of 1

================================================================================
                        DEPLOYMENT STEPS
================================================================================

1. Get GROQ_API_KEY
   → Visit https://console.groq.com
   → Create account (free, 2 minutes)
   → Copy API key (format: gsk_xxxxx...)

2. Update local environment
   → echo "GROQ_API_KEY=gsk_xxxxx..." > .env.local
   → npm install

3. Test locally
   → npm run dev
   → Open http://localhost:3000
   → Type "¿Argentina vs Marruecos?"
   → Verify response appears

4. Deploy to Vercel
   → git add -A
   → git commit -m "feat: IA-Zak v3.0 with Groq chat, Kelly, i18n"
   → git push origin main
   → Vercel auto-deploys

5. Add env vars to Vercel
   → vercel env add GROQ_API_KEY
   → Paste your key
   → Production auto-redeploys

6. Test production
   → https://mundial2026.vercel.app
   → Type in chat
   → Verify /api/chat endpoint works
