CARACTERÍSTICAS TÉCNICAS
================================================================================

1. CORRELACIÓN INTELIGENTE
   ✅ Home Win + Over 2.5 → correlación positiva (1.05-1.10x)
   ✅ Home Win + Under 2.5 → correlación negativa (0.85-0.90x)
   ✅ BTTS + Over 2.5 → muy correlacionados (1.08-1.15x)
   ✅ Ajustes documentados en cada parlay

2. KELLY CRITERION PARA PARLAYS
   ✅ Fórmula: P_combined = P1 × P2 × P3 × correlation_factor
   ✅ Edge calculado: (P_combined × O_combined) - 1
   ✅ Kelly %: (Edge × P_combined) / O_combined
   ✅ Monto en ₡: Kelly % × Bankroll (cap ₡50,000)

3. TRES PERFILES DE RIESGO
   ✅ CONSERVADOR (Kelly 3-5%, 2 eventos max, probabilidad alta ~25-30%)
   ✅ MODERADO (Kelly 5-8%, 2-3 eventos, probabilidad media ~20-25%)
   ✅ AGRESIVO (Kelly 10-15%, 3-4 eventos, probabilidad baja ~15-18%)

4. INTEGRACIÓN FERXXXA
   ✅ Validar parlays contra opinión comunitaria
   ✅ Detectar arbitrage (si comunidad diverge)
   ✅ Ajustar confianza según consensus
   ✅ Comparar parlays con trending bets

5. FORMATO JSON ESTRUCTURADO
   ```json
   {
     "name": "Conservative - Argentina Win + Under 2.5",
     "events": [
       {"market": "1x2", "prediction": "home_win", "probability": 0.65, "odds": 1.75},
       {"market": "over_under", "prediction": "under_2.5", "probability": 0.45, "odds": 1.95}
     ],
     "combined_probability": 0.26,
     "combined_odds": 3.41,
     "correlation_adjustment": "0.85x (negative)",
     "kelly_percentage": 3.8,
     "bankroll_amount_colones": 1900,
     "risk_profile": "conservative",
     "reasoning": "Explicación detallada con citas de fuentes..."
   }
   ```
