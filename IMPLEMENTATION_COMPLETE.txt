QUICK START GUIDE
================================================================================

STEP 1: Install Dependencies
  $ cd ~/mundial2026
  $ npm install

STEP 2: Set Environment Variables
  export DORADOBET_USER=your_username
  export DORADOBET_PASS=your_password
  export CRON_SECRET=your_secure_token
  export DATABASE_URL=postgresql://...
  export NODE_ENV=production

STEP 3: Test Locally
  $ npm run dev
  # In another terminal:
  $ curl -H "Authorization: Bearer $CRON_SECRET" \
    http://localhost:3000/api/ferxxxa-markets

STEP 4: Configure Vercel (vercel.json)
  {
    "crons": [
      {
        "path": "/api/ferxxxa-markets",
        "schedule": "*/5 * * * *"
      }
    ]
  }

STEP 5: Deploy
  $ vercel --prod

STEP 6: Monitor
  $ vercel logs --follow
